# setAppFull/设置应用全屏
set app to full screen run.<br>
将应用突破次元壁，覆盖异形屏幕限制全面显示_(:з」∠)_<br>
强制让异形屏幕的手机运行游戏时可以全屏显示无黑边<br>
部分功能借鉴了[BCRfullscreen](https://github.com/KitsunePie/BCRfullscreen)<br>
